# Site vitrine IADO.IO

## Contenu du projet

- `index.html` : structure complète du site
- `styles.css` : design responsive
- `script.js` : menu mobile, animations et envoi Formspree

## Ouvrir le site en local

Double-cliquez sur `index.html`.

Pour tester dans de meilleures conditions, utilisez par exemple l’extension
**Live Server** dans Visual Studio Code.

## Configurer Formspree

1. Créez un formulaire sur Formspree.
2. Copiez l’adresse fournie, par exemple :

   `https://formspree.io/f/abcdwxyz`

3. Dans `index.html`, recherchez :

   `https://formspree.io/f/VOTRE_IDENTIFIANT`

4. Remplacez-la par votre adresse Formspree.

Le JavaScript envoie le formulaire sans recharger la page et affiche un message
de réussite ou d’erreur.

## Déploiement gratuit

Ce site peut être déployé sur :

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

Comme Formspree gère la réception des formulaires, aucun serveur personnel n’est
nécessaire dans cette première version.
